# Demo Repository for Repodoc Viewer

This folder is intentionally designed for product demonstrations.

What to demo:
- Ask the assistant to summarize architecture from docs and code.
- Search for TODO/FIXME items and propose changes.
- Open and preview HTML and SVG files.
- Inspect CSV and log files for trends.

Suggested prompts:
- "Explain how release risk is calculated."
- "Find all TODO items and group by severity."
- "What does the deployment script validate before deploy?"
- "Summarize failed jobs from app.log."
- "Show me how to apply a fix to a file outside this project." (Note: External writes are disabled by default and require manual opt-in).
